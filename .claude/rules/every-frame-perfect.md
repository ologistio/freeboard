# Every Frame Must Be Explainable

Treat every rendered frame as a finished UI state, including loading, transitions, and animations.

* Never flash incomplete or unstyled content.
* Avoid layout shifts as content loads.
* Keep related UI elements visually and logically synchronised.
* Ensure animations remain coherent between their start and end states.
* Every screenshot taken at any moment should show an intentional, explainable state.

A polished interface builds trust. Do not let technical limitations produce confusing or misleading frames.
