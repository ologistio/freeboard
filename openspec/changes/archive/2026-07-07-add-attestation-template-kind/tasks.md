## 1. Config kind: parse and model (feat(gitops): core model + loader)

- [x] 1.1 In `src/Freeboard.Core/GitOps/ConfigModel.cs` add `KindAttestationTemplate` to `GitOpsSchema`; add the three YAML-bound records `AttestationField`, `QuizItem`, and `AttestationTemplate` as init-property records with a public parameterless constructor and `= string.Empty` / `= []` init defaults, matching the existing kinds exactly (for example the `EvidenceCollector` record). They MUST NOT be positional records: YamlDotNet's default object factory needs a public parameterless constructor, positional-only records have none, and `ConfigLoader` registers no custom `IObjectFactory` (also, `List<string> Options = []` is not a valid positional default parameter). The nested value records (fields and quiz items carry no `apiVersion`/`kind` of their own):

  ```csharp
  public sealed record AttestationField
  {
      public string Id { get; init; } = string.Empty;
      public string Label { get; init; } = string.Empty;
      public string Type { get; init; } = string.Empty;
      public List<string> Options { get; init; } = [];
  }

  public sealed record QuizItem
  {
      public string Id { get; init; } = string.Empty;
      public string Prompt { get; init; } = string.Empty;
      public List<string> Options { get; init; } = [];
      public string Answer { get; init; } = string.Empty;
  }
  ```

  and the template record:

  ```csharp
  public sealed record AttestationTemplate
  {
      public string ApiVersion { get; init; } = string.Empty;
      public string Kind { get; init; } = string.Empty;
      public string Id { get; init; } = string.Empty;
      public string Title { get; init; } = string.Empty;
      public string Control { get; init; } = string.Empty;
      public string Type { get; init; } = string.Empty;
      public string Body { get; init; } = string.Empty;
      public List<AttestationField> Fields { get; init; } = [];
      public List<QuizItem> Quiz { get; init; } = [];
      public string PassMark { get; init; } = string.Empty;
  }
  ```

  `PassMark` is raw authored text (matching `EvidenceCollector.Threshold`) so a malformed value is a clean validation diagnostic, not a YamlDotNet binding error. Add `List<AttestationTemplate> AttestationTemplates { get; init; } = []` to `GitOpsConfig`. This YamlDotNet parameterless-constructor constraint applies to the Core config-model records only; the read-model/wire-record copies used solely by `System.Text.Json` (`ComplianceReadModels` `QuizItemView`/`AttestationTemplateRow`, CLI `ApiQuizItem`/`ApiAttestationField`/`ApiAttestationTemplate`) MAY remain positional records.
